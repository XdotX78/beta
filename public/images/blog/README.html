Mystery Blog Images

This page displays the SVG files created for the Mystery Blog header. For production use, these should be converted to PNG or JPG files for better browser compatibility.

mysteries-background.svg

Description: This image serves as the main background for the Mystery Blog header. It features a dark purple starry sky with a subtle constellation pattern and a distant moonlight effect.

Current Path: /images/blog/mysteries-background.svg

Production Recommendation: Convert to JPG and use as /images/blog/mysteries-background.jpg

fog-overlay.svg

Description: This image provides a misty fog effect to overlay on top of the main background, creating an atmospheric mystery-themed header.

Current Path: /images/blog/fog-overlay.svg

Production Recommendation: Convert to PNG with transparency and use as /images/blog/fog-overlay.png

How to Convert for Production

You can convert these SVG files to PNG/JPG using various tools:

  • Browser: Open the SVG in a browser, take a screenshot, and crop as needed
  • Inkscape: Open the SVG and export as PNG/JPG
  • ImageMagick: Use the command line: 
    convert mysteries-background.svg mysteries-background.jpg
convert fog-overlay.svg fog-overlay.png
  • Online Converters: Various websites can convert SVG to PNG/JPG

Note: The current blog code is looking for JPG/PNG versions of these files. If you wish to use the SVG files directly, update the paths in gaia/src/app/blogs/mysteries/page.tsx.
